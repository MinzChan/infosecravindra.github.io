Flutter Getting Started

Introduction

Flutter is a framework for writing UI, it is not a programming language, it uses dart as programming language, you can learn more about dart at Dart Website. I’d advice you to invest some time in learning dart, you’ll be able to understand everything faster by mastering dart basics.

Environment Setup

To start flutter development we need Android SDK and Flutter SDK, Follow official guide to setup your environment Flutter Installation

Basic Principal to start learning

  • In flutter everything a widget, in the beginning it feel like HTML where you have nested div inside a div but it gets better with time.
  • There are two type of widget stateless and stateful.
  • Stateless widgets are static, they can’t change.
  • Stateful widgets are dynamic which means they can be changed programmatically or by user inputs.
  • Stateful widget extends Statefulwidget and returns state object
  • State object extends State<Widget> and return UI that will be rendered.
  • State object overrides build method.

Conclusion

This list is very small and it is all you need to write UI with flutter.
