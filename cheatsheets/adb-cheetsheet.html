ADB Cheatsheet

A lot of usefull adb commands

List all connected devices

adb list devices

adb connect using device IP

adb connect 192.168.43.1

Check Multiuser Support

adb shell pm list max-users

Set/Hack Multiuser Support

adb shell setprop fw.max_users = 5

I don’t remember

adb shell “dumpsys package | grep -A1 ‘userId=UID’”
